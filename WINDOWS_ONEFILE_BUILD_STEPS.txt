HL Intelligence one-file Windows build
=====================================

After pulling the repo, run the commands from the repository root.

Windows PowerShell
------------------

  npm ci
  $env:HL_WINDOWS_DOWNLOADS="[replace with destination folder for exe file]"; npm run package:win

Example:

  $env:HL_WINDOWS_DOWNLOADS="C:\Users\YourName\Downloads"; npm run package:win

WSL
---

  npm ci
  HL_WINDOWS_DOWNLOADS="[replace with destination folder for exe file]" npm run package:win

Example:

  HL_WINDOWS_DOWNLOADS="/mnt/c/Users/YourName/Downloads" npm run package:win

Output
------

The command creates one portable Windows executable named:

  HL Intelligence.exe

If HL_WINDOWS_DOWNLOADS is not set, the default output path is:

  release/windows-portable/HL Intelligence.exe
